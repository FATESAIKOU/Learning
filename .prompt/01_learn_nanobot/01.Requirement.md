# nanobot 學習專案

## 目的
我需要學習 nanobot，理解它的用法，並且最後產出一個可以隨意部署的 dockerfile/docker-compose fil

## 預期結果

- 一個程式實現規劃
- 多個作業進度報告（每個步驟一份）
- 一份可部署的 nanobot 程式碼

## 詳細指示

- 預期使用 Openapi Codex OAuth 做 LLM 認證(然後預設使用其中最便宜的模型)
- 預期最終連線到 Telegram
- 程式碼放在 `01_nanobot_dockerize/result`
- 所有報告放在 `01_nanobot_dockerize/report`

## 步驟

1. 理解＆詳細化需求(輸出學習計劃)
2. 開發 openai-codex-oauth-proxy(主程式 & 測試程式碼各一份)
3. 測試 openai-codex-oauth-proxy
4. 下載 nanobot 使用 docker 做最基本包裝 啟動
5. 嘗試驗證連接 openai / telegram
6. 整理理解連結 openai / telegram 所需的基本手續 以及連結之後需保持的檔案
7. 規劃/設計初期配置用以及實際提供服務用的 dockerfile/dockercomposefile 檔案
8. 實現初期配置用 dockerfile/dockercomposefile ＆ 測試
9. 實現實際提供服務用 dockerfile/dockercomposefile ＆ 測試

※ 所有步驟都需要經過我 Review, 否則不准進下一步
※ 為保持User理解你的進度並且Review, 不准同時進行多步
※ 跟User對話與程式註解用中文 只有程式碼本身用英文
